Front End Developer Assessment

This project is designed to be a quick but realistic example of front-end web challenges we deal with, and a way for you to show us how you go about working with those challenges.

The Project

Client: Mission Bicycle Company

Project: Redesign their site. Create a one page web site that allows a customer to browse the product catalog.

The Requirements

  • The site must be responsive. The client wants to support a wide range of devices.
  • The product catalog should be retrieved from Mission Bicycle's product catalog service. This service can be accessed via a GET request to http://seq-front-end-assessment.s3-website-us-west-2.amazonaws.com/catalog.json. This JSON file should contain all product information and images associated with products.
  • The client wishes to use Googles Montserrat and Playfair Display web fonts.
  • Font Awesome web fonts should be used for iconography.
  • All code should be checked into a git repository. Bitbucket or Github will work.

Build Assets

  • Adobe Photoshop PSD
  • Font Files for Photoshop
  • PNG Files
  • Prototype

Expectations

We do not expect you to spend more than a couple of hours on this. Get as far as you can within a couple of hours and share your git repository with us.

Our intent is to see how you go about setting up the project. How you structure the project and what tools do you choose to use for example. This will help us get an idea on how you work and will provide us with talking points during our interview.
